# 产品需求文档：Node-RED低代码测绘实训监测平台

## 1. 产品概述

本产品旨在构建一个基于Node-RED的低代码实训监测平台，用于连接和控制全站仪设备。系统面向高校教学（老师/学生）及科研人员，通过可视化编程降低设备控制门槛，支持从基础测量到复杂科研流程的自定义开发。

## 2. 用户角色与权限

### 2.1 管理员（老师）

- 拥有最高权限
- 可创建教学任务
- 可配置系统参数
- 可查看所有数据

### 2.2 普通用户（学生）

- 可登录系统执行测量任务
- 通过低代码界面控制设备
- 可导出个人报表

### 2.3 开发者（科研人员）

- 可利用Node-RED底层能力开发自定义功能节点

## 3. 核心功能需求

### 3.1 系统基础模块

**用户登录**

- 支持账号密码验证
- 区分管理员与普通用户权限

**多模态连接**

- 支持线缆、蓝牙、网络通信
- 远程网络连接需输入IP地址、端口号及设备波特率

**结果导出**

- 支持将测量结果导出为标准报表格式

### 3.2 设备控制与测量模块

**任务创建**

- 用户可输入工作项目名称
- 支持调用全站仪拍照功能，记录项目环境影像

**目标搜索与锁定**

- 控制设备扫描周围棱镜
- 利用全站仪相机系统视觉辅助，自动转向并锁定下一个目标棱镜

**一键测量**

- 集成测角、测距功能
- 可配置测量模式及重复测量次数

### 3.3 AI智能辅助模块

**操作指导**

- 提供实时步骤指引

**疑难解答**

- 解答设备操作及测量流程中的问题

**数据分析**

- 辅助用户进行测量数据的处理与分析

### 3.4 设站功能（核心测量逻辑）

系统提供图形化界面模拟全站仪设站操作，实时显示设备状态（电量、Hz/V角度、时间）。

**设站方式：**

- **后方交会：** 测量两个或更多已知目标点，计算测站坐标和定向
- **已知后视点：** 基于已知点进行定向
- **高程传递：** 用于高程数据的传递与校准
- **其他定向：** 支持方位角设置、线定向、物定向等模式

### 3.5 测量作业界面

**实时控制**

- 显示Hz（水平角）、V（垂直角）、平距等实时数据

**搜索策略**

- **超级搜索过滤器：** 提升搜索效率
- **自动照准：** 自动瞄准目标
- **跟踪模式：** 支持"搜索并跟踪"及"等待并跟踪"动态目标

## 4. 课程与进阶开发体系

### 阶段一：基础实训（面向学生）

**目标：** 掌握测绘流程

**方式：** 学生无需编写代码，直接使用系统内封装好的Node-RED节点（如"设站节点"、"测量节点"）

**操作：** 拖拽节点并连线，搭建工作流，实现对全站仪的正确操控

### 阶段二：高级开发（面向科研/教师）

**目标：** 实现特殊功能定制

**方式：** 允许用户基于Node-RED框架开发自定义功能节点

**应用场景：**

- 道路检核与放样
- 铁路检核与放样
- 隧道检核与放样

## 5. 界面布局原型摘要

**登录页**

- 账号、密码输入框

**连接页**

- 连接方式下拉菜单
- IP、端口、波特率配置

**设站页**

- 模拟全站仪界面
- 含角度数据显示
- 设站方式选择及图文说明

**测量页**

- 相机视图辅助
- 实时距离角度显示
- 测量/保存/换页功能按钮
