# station-setup — 需求规格

## 概述

提供图形化全站仪设站界面，支持后方交会、已知后视点、高程传递、其他定向四种设站方式，实时显示设备状态数据。

## 项目信息

- 项目名: node-red-project
- 架构类型: Full-stack TypeScript Monorepo (pnpm + Turborepo)

## 需求版本

| 日期       | 版本 | 说明     |
| ---------- | ---- | -------- |
| 2026-06-13 | v1   | 初始需求 |

## 用户故事

- 作为学生，我想要通过图形界面完成全站仪设站，以便掌握测绘基本流程
- 作为用户，我想要实时看到设备角度/电量/时间状态，以便监控设备工作状态
- 作为管理员，我想要查看所有学生的设站记录，以便评估实训效果

## 功能需求

1. [F-001] 实时状态面板：电量、Hz（水平角）、V（垂直角）、当前时间，通过 WebSocket 推送
2. [F-002] **后方交会**：输入 ≥2 个已知点坐标，逐点照准测量，计算测站坐标与定向角
3. [F-003] **已知后视点**：输入测站坐标 + 后视点坐标，照准后视点完成定向
4. [F-004] **高程传递**：输入已知高程点，进行高程数据传递与校准
5. [F-005] **其他定向**：支持方位角设置、线定向、物定向三种子模式
6. [F-006] 设站结果（测站坐标、定向角、残差）保存到数据库，与当前工作任务关联
7. [F-007] 已知点库：用户可预存常用已知点，设站时快速选入

## 非功能需求

- 精度: 坐标/角度计算结果保留 4 位小数
- 安全: 设站数据与用户绑定，学生只能查看自己的记录
- 性能: 实时角度数据刷新频率 ≥ 1Hz

## 验收标准

- [ ] [AC-001] 完成后方交会流程：输入已知点 → 测量 → 查看计算结果（测站坐标 + 残差）
- [ ] [AC-002] Hz/V 角度数据实时更新，与设备实际角度一致
- [ ] [AC-003] 设站结果持久化，下次登录后可在历史记录中查看
- [ ] [AC-004] 已知点库可增删改，设站时下拉选择即可填入坐标

- `packages/db` — 新增 stations、known_points 表
