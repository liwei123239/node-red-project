# measurement — 需求规格

## 概述

提供完整的测量作业界面，包含任务创建、目标搜索与锁定、一键测量、实时数据显示、跟踪模式及报表导出。

## 项目信息

- 项目名: node-red-project
- 架构类型: Full-stack TypeScript Monorepo (pnpm + Turborepo)

## 需求版本

| 日期       | 版本 | 说明     |
| ---------- | ---- | -------- |
| 2026-06-13 | v1   | 初始需求 |

## 用户故事

- 作为学生，我想要创建测量任务并拍照记录环境，以便建立完整的测量档案
- 作为用户，我想要一键触发测量并实时看到角度距离数据，以便高效完成测量工作
- 作为用户，我想要设备自动搜索并锁定棱镜目标，以便减少人工操作
- 作为用户，我想要导出标准测量报表，以便提交成果

## 功能需求

1. [F-001] 任务创建：输入工作项目名称，调用设备拍照并保存环境影像
2. [F-002] 实时测量数据面板：Hz（水平角）、V（垂直角）、斜距/平距，WebSocket 实时推送
3. [F-003] 目标搜索：控制设备扫描周围棱镜，列出候选目标
4. [F-004] 自动照准（ATR）：选择目标后设备自动转向并锁定棱镜
5. [F-005] 一键测量：集成测角 + 测距，可配置测量模式（精测/粗测）和重复次数
6. [F-006] 搜索策略：超级搜索过滤器、自动照准、搜索并跟踪、等待并跟踪
7. [F-007] 测量结果列表：显示所有测量点，支持删除和备注
8. [F-008] 结果导出：将测量成果导出为 CSV 或 PDF 标准报表

## 非功能需求

- 性能: 实时数据推送延迟 < 200ms
- 安全: 测量数据与创建任务的用户绑定
- 可用性: 未完成设站时，测量功能禁用并提示先设站

## 验收标准

- [ ] [AC-001] 创建任务后，相机视图显示，拍照图片保存并在任务详情中显示
- [ ] [AC-002] 点击「搜索目标」后设备开始扫描，候选棱镜列表出现
- [ ] [AC-003] 选择目标后设备自动照准，Hz/V 数据更新到目标方向
- [ ] [AC-004] 点击「测量」后数据出现在结果列表中
- [ ] [AC-005] 导出 CSV 文件内容正确，包含所有测量点数据
